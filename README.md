# Microfluidic Chip for 6-Nitrochrysene Detoxification Study

## Overview
This project presents a microfluidic chip designed to support a lung-cell-based investigation of 6-nitrochrysene detoxification. The design was developed as an engineering platform for comparing treated and untreated conditions in a controlled, repeatable microfluidic environment.

## Background
6-nitrochrysene is a hazardous polycyclic aromatic compound with biological relevance in toxicology research. Because the project focuses on a lung-cell model, the chip was designed to support a comparison between a treated lane exposed to 6-nitrochrysene and an untreated control lane containing the same cell type and culture medium.

## Design Goals
The design goals were:
- Create two matched experimental lanes for direct comparison.
- Keep chamber dimensions consistent between conditions.
- Include clear inlet and outlet access points.
- Add smooth transitions between channels and chambers.
- Make the chip visually and structurally clear for documentation and future fabrication.

## Chip Layout
The final design uses two parallel lanes with repeated chamber regions. Each lane is intended to represent one experimental condition:
- Untreated lane: lung cells with standard culture medium.
- Treated lane: lung cells with culture medium containing 6-nitrochrysene.

The design includes inlet and outlet ports sized to support practical loading and access, along with reservoir-like openings at the chip edges.

## Why These Choices Were Made
The parallel-lane layout was chosen because it provides a direct treated-vs-untreated comparison without mixing the conditions. Matching chamber geometry ensures that differences in cell response can be attributed to treatment rather than device asymmetry. The reservoir features help make the device easier to load and visually communicate the fluidic entry and exit points.

## Engineering Relevance
This project combines CAD modeling, experimental planning, and biomedical engineering design. It demonstrates the process of translating a toxicology question into a microfluidic platform with controlled geometry and a clear comparative layout.

## Future Improvements
Possible next steps include:
- Adding a more detailed biological readout plan.
- Modeling flow behavior or diffusion.
- Refining the design for fabrication.
- Developing a more advanced membrane-based lung-chip version.
